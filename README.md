# Spring Java with PostgreSQL

Project sederhana Java yang terhubung ke PostgreSQL untuk mengelola data.

## Fitur
- Tambah title dan description
- Lihat semua data

## Teknologi
- Spring Boot
- Java
- PostgreSQL
- Redis
- JUnit

## Cara Menjalankan
1. Import `database/schema.sql` di PostgreSQL
2. Jalankan `Main.java`
